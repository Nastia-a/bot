# 🎬 YouTube Downloader Telegram Bot

Telegram бот для скачивания видео и аудио с YouTube с выбором качества.

## ✨ Возможности

- 📥 Скачивание видео в качестве 360p, 720p, 1080p
- 🎵 Скачивание аудио в форматах MP3 и M4A
- 📊 Информация о видео (название, длительность)
- 🚀 Быстрая загрузка файлов
- 📱 Простой интерфейс с inline-кнопками

## 🚀 Быстрый старт

### Предварительные требования

- Python 3.8+
- FFmpeg
- Telegram Bot Token (получить у [@BotFather](https://t.me/botfather))
